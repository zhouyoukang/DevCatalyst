## Search Strategy

### Priority Order
1. **Official docs** — `site:docs.xxx.com`
2. **GitHub repos** — Example code and best practices
3. **Stack Overflow** — Common Q&A
4. **Tech blogs** — In-depth analysis and tutorials

### Search Templates
```
# Find API usage
"<library> <function> example usage 2025"

# Find error solutions
"<error message>" site:stackoverflow.com OR site:github.com

# Find best practices
"<technology> best practices 2025 2026"

# Find alternatives
"<tool> vs <alternative> comparison 2025"
```

### After Learning (Must Do)
- Verify information recency (prefer last 2 years)
- Write valuable findings to Memory (create_memory)
- Consider creating new Skill for reusable knowledge
- Cross-validate with multiple sources
